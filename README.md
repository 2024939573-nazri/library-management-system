# 📚 Library Management System

A web-based Library Management System developed using HTML, CSS, JavaScript, and Bootstrap 5. This project was created as an Individual Assignment to demonstrate the implementation of a simple library management interface with responsive design and interactive dashboard features.

## 🎯 Project Overview

The Library Management System provides a user-friendly platform for managing library-related information such as books, categories, borrowing records, and user profiles. The system includes a modern dashboard with statistical summaries and data visualization using Chart.js.

## ✨ Features

- User Login Interface
- Dashboard Overview
- Book Management Module
- Borrow Records Management
- Categories Management
- User Profile Page
- Responsive Design using Bootstrap 5
- Interactive Dashboard Chart
- Modern UI with Background Images and Animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Chart.js
